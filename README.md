# wade-usa.com

This repository hosts the code for wade-usa.com, my personal website. This site will showcase various applications I learn and build, and will also feature a travel blog.

## Project Overview

* **Frontend:** Built with React
* **Backend (CMS):** Strapi (Headless CMS)
* **Database:** PostgreSQL
* **Containerization:** Docker
* **Deployment:** DigitalOcean Droplet (Ubuntu 24.04)

## Getting Started (Local Development)

### Prerequisites

Make sure you have the following installed on your local machine:
